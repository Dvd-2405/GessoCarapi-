# Gesso Carapiá 🏗️

> *"Do teto ao detalhe, a gente tem."*

## O que é esse projeto

Esse site foi construído por mim, David, como uma homenagem ao meu pai — **Domingos Savio** — e à empresa que ele ergueu com as próprias mãos.

Meu pai começou a trabalhar com gesso aos **14 anos**. Não por escolha de carreira, mas por necessidade. Ele aprendeu o ofício na prática, no suor, no dia a dia — e nunca mais parou.

Em **1990**, com muito esforço e coragem, ele fundou a **Gesso Carapiá**. De lá pra cá, já são mais de **35 anos** servindo clientes, sustentando família e construindo, literalmente, os tetos de muita gente.

Eu cresci vendo meu pai trabalhar. Vi ele sair cedo, chegar tarde, resolver problema, atender cliente, carregar peso, fazer conta. Ele fez — e faz — de tudo por mim e pela minha família. Ele é o cara mais brabo do gesso que eu conheço.

Quando comecei a estudar desenvolvimento web, sabia que o primeiro projeto real que eu entregasse para alguém precisava ser pra ele.

Esse site é isso: o meu primeiro presente de dev pro meu pai.

---

## Tecnologias utilizadas

- HTML5
- CSS3 (variáveis, flexbox, grid, responsividade)
- JavaScript (carrossel, filtro de produtos, dropdown)
- GitHub Pages (deploy)

---

## Funcionalidades

- Carrossel de fotos da loja e serviços com autoplay
- Seção de serviços com carrossel próprio
- Catálogo de produtos com filtro por categoria
- Links diretos para WhatsApp com mensagem automática
- Dropdown de localização e redes sociais
- Layout totalmente responsivo
